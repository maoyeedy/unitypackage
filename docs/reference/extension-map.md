# File tree icon mapping

## Image

| Extension              | MIME type           |
|------------------------|---------------------|
| png                    | image/png           |
| jpg, jpeg              | image/jpeg          |
| gif                    | image/gif           |
| bmp                    | image/bmp           |
| apng                   | image/apng          |
| avif                   | image/avif          |
| webp                   | image/webp          |
| svg                    | image/svg+xml       |

## Fallback
